## Сделано — хроника v1→v4

- **v1** — плоский showcase компонентов, спокойные серые тона, 1 светлая + 3 тёмные темы-кандидата.
- **v2** — парящий layout: 2 пробных варианта (A «утопленный холст» / B «плавающие карточки»), токены
  поверхностей `--surface-0/1/2`, primary→асфальт (chroma≈0), тёмная сведена к **graphite-only**.
  Добавлены bar-chart, sparkline, segmented/pill-nav, avatar-stack, icon-btn, date-chip, empty, toast,
  hover-elevation. Зафиксирован фикс контраста кнопок в dark.
- **v3** — выбран **гибрид**: утопленный chrome (sidebar + хэдер без фона) + плавающие карточки.
  Хэдер сделан **2-этажным** (floor-1 глобальный: workspace/навигация/поиск/тема/юзер; floor-2 страница:
  заголовок+действия). Тема-тумблер переехал в floor-1. Destructive подобран → **чистый красный**
  `oklch(0.58 0.21 25)`; `--destructive-bg` через `color-mix` (следует за токеном). Живой свитч оттенков в demo.
- **v4** — **точечный акцент** mint `#78FC90` через один токен `--brand` (picker-ready: `-fg/-bg/-strong`,
  `--ring`=brand). Акцент дозированно: активный пункт сайдбара, пики графиков, focus-ring, статус-точки/+N/badge.
  Варианты `.btn-accent`/`.btn-accent-soft`, `.badge-accent`, `.dot`/`.has-dot`. Demo dev-панель: Accent on/off
  и CTA mint/asphalt для живого сравнения. **CTA по умолчанию — асфальт** (mint опционально).

## Принятые решения

- Тёмная тема — **только graphite** (тёплый серый, не чёрный).
- **primary = асфальт**, серый без синевы (chroma ≈ 0); акцент монохром, цвет только смысловой.
- **destructive = `oklch(0.58 0.21 25)`** (чистый красный), зафиксирован.
- **Акцент = один токен `--brand`** (`oklch(0.88 0.2 150)` ≈ #78FC90); смена в будущем = одна переменная.
- **2 яруса глубины** (фон/chrome `--surface-0` → карточки `--surface-2`); `--surface-1` — резерв. Chrome утоплен, карточки парят (`--shadow-card-float`).
- Layout: утопленный sidebar + 2-этажный хэдер; активный «ты здесь» — только сайдбар (пилюли нейтральны).
- CTA — асфальт по умолчанию.

## Грабли / выводы

- В dark **secondary-кнопки сливались** с карточкой → `--secondary` сделан светлее карточки + `--border-strong`
  на secondary/outline, ghost-hover на `--accent`. Контраст проверять на `--surface-2`, не на голом фоне.
- `*-bg` для destructive/brand — через **`color-mix(in oklab, … , var(--surface-2))`**, чтобы подложки
  автоматически следовали за основным токеном (важно для будущего picker'а).
- Demo dev-панель должна стоять в DOM **до** `<script>` — иначе `getElementById` = `null` и обработчики не вешаются.
- Мята **светлая** → не использовать как мелкий текст на светлом фоне (контраст); для текста — затемнённый
  микс `--brand`+`--fg` (`.badge-accent`, `.btn-accent-soft`).
- Два зелёных (success + mint) рядом спорят → не ставить `badge-success` и `badge-accent` вплотную.
